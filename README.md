# Simple Interest Calculator

This project is a Simple Interest Calculator created using a Bash script.

## Description
The calculator computes simple interest using the following inputs:
- Principal amount
- Rate of interest
- Time period

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script `simple-interest.sh` and enter the required values when prompted.

## Example
Principal = 1000  
Rate = 5  
Time = 2  

Simple Interest = 100
